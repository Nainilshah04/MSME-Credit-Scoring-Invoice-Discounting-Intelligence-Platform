# MSME Credit Scoring + Invoice Discounting Intelligence Platform

A production-grade fintech ML platform for alternative credit scoring of MSMEs and invoice risk assessment for invoice discounting.

## 🎯 Project Overview

This platform addresses two critical fintech challenges:

1. **Credit Scoring Model**: Alternative credit assessment for Micro, Small & Medium Enterprises (MSMEs) using GST patterns, UPI transaction data, and business history
2. **Invoice Risk Model**: Real-time risk assessment of invoices for invoice discounting/factoring

## 🏗️ Tech Stack

- **Language**: Python 3.9+
- **ML Libraries**: scikit-learn, pandas, numpy
- **Database**: SQLite (Development), PostgreSQL-ready (Production)
- **Visualization**: matplotlib, seaborn
- **Testing**: pytest
- **Version Control**: Git
